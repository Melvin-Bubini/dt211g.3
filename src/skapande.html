Information om utveckling

Min arbetsprocess för denna uppgift var så att jag började med att skapa npm-projekt med "npm init". Därefter så installerade jag parcel med "npm install parcel --dev-save". När den var nedladdad så la jag min node_modeules i ett .gitignore fil som jag skapade i samband med det. Sedan gick jag in i min package.json fil och skrev egna script och skrev till en source med index.html filen som start. När jag hade skrivit klart min package.json fil så skapade jag en src mapp och i den mappen så skapade jag en fil för html, css och javascript. Därefter så började skapandet av demowebbplatsen, vilket jag startade med att skapa en index.html fil. Under tiden som jag utvecklade webbplatsen så gjorde jag regelbundna commits till ett Github-repo som jag hade kopplat till denna uppgift. Det som är auotomatiserat för den här webbplatsen är storleken och formatet som bilerna visas i. Bilderna är alla i jpeg-format och olika storlekar. När de sedan visas på sidan är de istället alla i webp-format och i samma storlek. Storleken på bilderna beror på skärmens storlek. Är skärmen större än 800px i bredd så visas bilderna i 800px i bredd. Är skärmen större än 401px i bredd så visas bilderna i 400px i bredd. Den sista regeln är att om skärmen är mindre än 400px i bredd så visas bilderna i 200px i bredd. Slutligen, skulle en webbläsare inte klara av att visa någon av dessa bilder så finns det en bild som den faller tillbaka på och visar då.

Information om publiceringen

För att publicera uppgiften så använde jag mig av Netlify. Jag gick in på Netlify för att publicera och väl där så använde jag Github för att publicera via den sidan. Jag valde då mitt repo som jag arbetat i för att publicera det. Därefter så fick jag välja inställningar för publicering och jag valde då att använda build command "npm run buil" och att publicera dist katalogen. Därefter så var webbsidan publicerad och klar.
